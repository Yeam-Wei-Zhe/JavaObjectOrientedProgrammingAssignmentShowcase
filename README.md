# Tuition Centre Management System

A desktop application developed in Java using NetBeans for managing students, tutors, schedules, and subjects in a tuition centre.

## Features

- Student Login
- View Schedule
- Update Student Profile
- Subject Registration
- File-based Database
- Role-Based Authentication

## Technologies

- Java
- Java Swing
- Object-Oriented Programming
- File Handling
